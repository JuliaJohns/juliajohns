Get an API subscription key

Before you can try the API, you need an API subscription key.

To get an API subscription key:

  1. Sign Up: Sign up for a developer portal account.

  2. Activate: Click the link in the verification email to activate your account.

  3. Subscribe: Subscribe to a product and an API:

    1. Go to the Products tab and choose a product.

    2. Choose Subscribe for the API you want to consume.

    3. Name your subscription (optional). You can have multiple API subscriptions per account.

    4. Click Confirm.

  4. Approval: You will receive an email when the Hypertherm API Team approves your subscription and issues your API key. In the meantime, feel free to preview the API.

  5. Get Key: Go to your account profile.
