View reports

You can access analytics reports through your account profile.

To view analytics reports:

  1. Go to your developer portal profile.

  2. Select Analytics reports on the right.

  3. Select a report category tab: At a glance, Usage, Health, or Activity.

  4. Filter your results by Date range, Product, API, and/or Operation.
    Note: Reporting is only available for the past 90 days. Reports cannot be exported at this time.

Report categories

The following report categories are available:

Report Available data
At a glance

  • Usage/Health overall:

    • Calls

    • Response time

    • Bandwidth

    • Errors

  • Activity data for top products, subscriptions, APIs, and operations
Usage

  • Calls (over time and by region)

  • Bandwidth (over time and by region)

 
Activity

  • Successful calls

  • Blocked calls

  • Failed calls

  • Other calls

  • Average response time

  • Bandwidth

 
Health

  • Status codes

  • Cache (Hit/miss)

  • API response time

  • Service response time
